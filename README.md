# pythonddos

cara download :
1. git clone https://github.com/dimss7/pythonddos
2. cd pythonddos
3. chmod +x ddospython.py
4. python2 ddospython.py

python2 hostname,port,jumlah serangan ddos
contoh :
  python2 120.29.226.121 80 100 ddospython.py
  
  tekan ENTER
